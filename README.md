# Multi-Domain Data Analytics Portfolio: Practical Data Wrangling

## Executive Summary
This repository contains the practical portfolio completed during my data science training program. The core objective of these projects was to apply hands-on Python programming to clean, manipulate, and analyze diverse, real-world datasets. 

To bridge the gap between data preparation and advanced analytics, I utilized Python, Pandas, and NumPy for all core data-wrangling phases, while leveraging AI-assisted tools to prototype the downstream Machine Learning models and structure the final reports. This approach allowed me to master data manipulation while learning how to interpret automated ML outputs.

---

## 1. Major Project: Multi-Variable Data Wrangling & Prototyping
* **Core Technologies:** Python, Pandas, NumPy
* **Methodology:**
  * **Data Ingestion & Cleaning:** Used Pandas to import various multi-topic datasets, handle structural inconsistencies, and manage missing values.
  * **Data Transformation:** Applied NumPy for numerical operations, array manipulations, and handling statistical outliers.
  * **AI-Assisted Modeling:** Utilized AI tools to generate the predictive modeling code and tables. Focused on interpreting the resulting metrics to understand model performance.

---

## 2. Minor Project: Exploratory Data Analysis (EDA) & Reporting
* **Core Technologies:** Python, Pandas
* **Methodology:**
  * **Descriptive Statistics:** Used Pandas to aggregate data, compute summary statistics, and isolate key behavioral trends across different datasets.
  * **Structured Reporting:** Generated comprehensive tables and data summaries to make raw data clean and query-ready.

---

## Technical Skills Demonstrated
* **Pandas:** DataFrame manipulation, merging/joining datasets, handling missing data, and group-by aggregations.
* **NumPy:** Mathematical operations, indexing, and handling multi-dimensional arrays.
* **AI Collaboration:** Using AI effectively to accelerate workflow documentation and prototype code structures.
